# JavaScript Basics

Welcome to my JavaScript learning repository! This project contains a collection of simple scripts and programs that I wrote to practice and understand the core concepts of JavaScript.

## What's Inside?

The files are numbered so you can easily follow along. Here are the topics covered:

- **Variables & Data Types:** Understanding `let`, `var`, `const`, and the different types of data in JS.
- **Strings & Conditionals:** Working with text and using `if/else` statements to make decisions.
- **Loops:** Using loops like `for` and `while` to repeat actions.
- **Functions:** How to write reusable code, including modern Arrow Functions.
- **Arrays:** Storing lists of items and working with them.
- **Objects, Maps & Sets:** Understanding how to store data using key-value pairs and unique collections.
- **Object-Oriented Programming (OOP):** Learning about Classes, Inheritance, Getters, and Setters.

## How to Run

You can run any of these files using [Node.js](https://nodejs.org/). 
Just open your terminal, navigate to this folder, and type:

```bash
node <filename>.js
```
For example: `node 1_let_var_const.js`
